\textbf{FELADATKIÍRÁS}\\
\end{center}

A piaci termékek elõállításánál gyakran a költségek kétharmadát a szoftverfejlesztés teszi ki. Ezért a beágyazott alkalmazásoknál egyre inkább megszokottá válik beágyazott operációs rendszerek használata. Az összetett hardverek alkalmazása, a kód újrahasznosíthatósága, a csoportban történõ fejlesztés és a multitaszk igénye szintén szükségessé teszi valamilyen operációs rendszer alkalmazását.


A feladat célja különbözõ operációs rendszerek megismerése, elõnyeik és hátrányaik kitapasztalása és egy ipari alkalmazáson keresztül való összehasonlítása.
A hallgató feladatának a következõkre kell kiterjednie:
\begin{itemize}
	\item Adjon áttekintést az elterjedt operációs rendszerek
	\begin{itemize}
		\renewcommand\labelitemii{o}
		\item felépítésérõl,
		\item mûködésérõl,
		\item elõnyeirõl,
		\item hátrányairól!
	\end{itemize}
	\item Különbözõ fejlesztõkártyák segítségével hozzon létre beágyazott operációs rendszeres alkalmazást!
	\item A feladat megoldása során használjon különbözõ komplexitású és erõforrás-igényû operációs rendszereket!
	\item Hasonlítsa össze az operációs rendszereket különbözõ szempontok alapján!
	\item A hallgató végezzen irodalomkutatást a teljesítménymutatók mérésének témakörében!
	\item Tegyen javaslatot az összehasonlítás alapját adó metrikákra és végezze el az összehasonlító teszteket!
	\item Adjon javaslatot az elemzett operációs rendszerek felhasználásának területeire!
\end{itemize}
