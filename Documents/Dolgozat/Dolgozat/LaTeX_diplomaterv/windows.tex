\section{Windows 10 IoT Core}
%----------------------------------------------------------------------------

A Windows~10~IoT~Core a Windows~10 kisebb erõforrással rendelkezõ eszközökre optimalizált változata. Az x86/x64 alapú rendszerek mellett az ARM mikroprocesszorokat tartalmazó eszközökön is futtatható. Szoftver fejlesztéséhez az \emph{Universal Windows Platform} (UWP) API használható.

Az UWP API széleskörû támogatással rendelkezik a perifériák terén, a különbözõ Microsoft szolgáltatások (mint például az \emph{Azure}) szintén használhatóak.
